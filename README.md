연속 tool 호출방법

우선 모델에게 요청을 보내면 ,모델은 도구 호출을 포함한 응답을 보냈는지 확인한다. 그리고 요청한 tool들을 직접 실행하여 그 결과값을 체크하여 모델에게 다시 전달한다. 모델은 결과를 보고 작업이 더 필요할지에 대한 여부를 확인한다. 이 과정을 모델이 더 이상 추가 호출이 필요 없을 것이라고 판단할 때까지 루프문을 이용하여 판단하다가, 더 이상 필요하지 않다고 판단하면 답을 출력하고 루프에서 탈출한다. 
이러한 과정을 통해 모델은 단순 검색 기능 뿐만 아니라 복잡한 요청에 대한 다른 요청들도 처리할 수 있게 된다.
